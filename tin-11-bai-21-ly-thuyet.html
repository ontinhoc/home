Tin 11 - Bài 21: Thực hành cập nhật và truy xuất dữ liệu các bảng

Bài học tập trung vào thao tác thêm, sửa, xóa dữ liệu trong bảng CSDL và cách truy xuất dữ liệu bằng giao diện hoặc câu truy vấn SQL.

Mục tiêu Nội dung chính Luyện tập

I. Mục tiêu bài học

  • Biết cập nhật dữ liệu (thêm, sửa, xóa) trong bảng CSDL.
  • Biết truy xuất dữ liệu bằng giao diện và bằng câu truy vấn SQL.
INSERT UPDATE DELETE SELECT SQL

II. Nội dung chính

1. Thêm mới dữ liệu vào bảng

  • Chọn bảng → thẻ Dữ liệu.
  • Thêm dòng mới bằng phím Insert hoặc chuột phải → Chèn hàng.
  • Nhập dữ liệu cho từng trường; trường AUTO_INCREMENT không cần nhập.

2. Chỉnh sửa dữ liệu trong bảng

Khi dữ liệu nhập bị sai:

  • Nháy đúp vào ô dữ liệu cần sửa.
  • Nhập lại giá trị đúng → Enter.

3. Xóa dòng dữ liệu trong bảng

  • Chọn dòng cần xóa (Shift: chọn nhiều dòng liền nhau, Ctrl: chọn nhiều dòng rời nhau).
  • Nhấn Ctrl + Delete hoặc nút Xóa.
  • Xác nhận OK để xóa.

4. Truy xuất dữ liệu từ bảng

  • a) Truy xuất đơn giản: Chọn bảng → thẻ Dữ liệu → xem toàn bộ dữ liệu.
  • b) Sắp xếp dữ liệu: Nháy vào tên trường (ví dụ: khóa chính). Nháy lại để đổi tăng dần / giảm dần.
  • c) Tìm kiếm – lọc dữ liệu: Chuột phải vào cột → Quick Filter. Dùng điều kiện LIKE "%ký_tự%". Có thể xóa bộ lọc khi không cần.

5. Truy xuất dữ liệu bằng câu truy vấn SQL

Cú pháp tổng quát:
SELECT danh_sách_trường
FROM tên_bảng
[WHERE điều_kiện]
[ORDER BY tên_trường ASC | DESC];
  • SELECT: chọn trường cần lấy.
  • FROM: bảng dữ liệu.
  • WHERE: điều kiện lọc.
  • ORDER BY: sắp xếp kết quả.

III. Luyện tập – Vận dụng

  • Cập nhật dữ liệu bảng khác (ví dụ: casi).
  • Truy xuất dữ liệu theo nhiều tiêu chí khác nhau.
  • Vận dụng với bảng Tỉnh/Thành phố trong CSDL quản lí.
