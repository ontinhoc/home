LÀM VIỆC VỚI HeidiSQL

HeidiSQL là công cụ quản lý cơ sở dữ liệu MySQL/MariaDB trực quan, miễn phí, hỗ trợ kết nối qua TCP/IP và SSH. Bạn sẽ tạo kết nối, thao tác trên bảng bằng giao diện đồ họa hoặc mở tab truy vấn để chạy câu lệnh SQL.

Mục tiêu Kết nối và thao tác dữ liệu
Kỹ năng Query, Backup, Restore
Phím tắt Ctrl+T   F9
1

Kết nối đến Server MySQL/MariaDB

Thiết lập phiên kết nối mới và điền đúng thông tin đăng nhập trước khi mở phiên làm việc.

  • Mở HeidiSQL và nhấn New để tạo một phiên kết nối mới.
  • Điền thông tin kết nối: Network type: MySQL (TCP/IP), Hostname/IP: localhost hoặc IP máy chủ, User/Password: tài khoản MySQL, Port: 3306.
  • Nhấn Open để kết nối.
Ghi nhớ: nếu chạy trên máy cục bộ, Hostname/IP thường là localhost.
2

Các thao tác thực hành cơ bản

Thực hiện các thao tác tạo database, tạo bảng, nhập liệu và chỉnh sửa cấu trúc.

  • Tạo Database: chuột phải vào tên server → Create new → Database.
  • Tạo Table (Bảng): chọn Database → chuột phải → Create new → Table. Thiết lập Column, Data type và Primary Key.
  • Nhập liệu (Data): chọn bảng → tab Data → nhấn nút + (Insert row).
  • Chỉnh sửa cấu trúc: tab Table cho phép thay đổi kiểu dữ liệu, thêm hoặc xóa cột.
3

Thực thi Truy vấn SQL (Query)

Viết và chạy các câu lệnh SQL trực tiếp trong tab truy vấn.

  • Nhấn Ctrl+T hoặc chọn tab Query để viết lệnh SQL (SELECT, INSERT, UPDATE, DELETE).
  • Nhấn F9 hoặc biểu tượng Play màu xanh để chạy câu lệnh.
  • HeidiSQL hỗ trợ autocomplete và hiển thị cấu trúc bảng để viết lệnh nhanh hơn.
Mẹo: các lệnh đã chạy thường được ghi lại trong lịch sử để bạn dùng lại nhanh.
4

Sao lưu và Khôi phục (Backup/Restore)

Bảo vệ dữ liệu bằng cách xuất và nhập file SQL.

  • Sao lưu (Export): chuột phải vào Database → Export database as SQL. Chọn bảng, cấu trúc, dữ liệu cần xuất rồi nhấn Export.
  • Khôi phục (Import): File → Load SQL file → chọn file .sql → nhấn F9 để chạy.
5

Quản lý kết nối

Tổ chức các phiên làm việc để chuyển đổi nhanh giữa môi trường Local, Production hoặc Development.

  • Có thể lưu nhiều Sessions khác nhau ngay trong cửa sổ kết nối ban đầu để chuyển đổi nhanh chóng.
