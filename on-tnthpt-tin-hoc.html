Gợi ý sử dụng

Quy trình nhanh để đưa hệ thống vào sử dụng với học sinh.

1. Chạy npm install và npm start trong thư mục server/.
2. Đổi mật khẩu quản trị bằng biến môi trường ADMIN_PASSWORD nếu cần.
3. Vào khu giáo viên để tạo đề hoặc nhập JSON.
4. Chuyển đề sang trạng thái phát hành.
5. Cho học sinh mở trang này và làm bài theo mã đề.
Trang ưu tiên data-api-base, có thể override tạm thời bằng tham số ?apiBase=https://.... Khi mở local, trang tự thử thêm http://127.0.0.1:3000; khi host online, trang sẽ thử thêm cùng origin nếu có proxy /api.
