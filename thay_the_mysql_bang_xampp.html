Thao tác với MySQL qua dòng lệnh (CLI) bao gồm các bước: kết nối (mysql -u root -p), Cách kết nối bằng MariaDB: mysql --user=”root” –passwword=””, quản lý database (CREATE/DROP), quản lý bảng (CREATE/DROP/ALTER), và thao tác dữ liệu (SELECT, INSERT, UPDATE, DELETE). Mọi câu lệnh SQL phải kết thúc bằng dấu chấm phẩy (;). 

1. Kết nối MySQL

# Kết nối với user root

mysql -u root -p     (MariaDB: mysql --user=”root” –passwword=””)

# Nhập mật khẩu khi được yêu cầu

2. Thao tác với Database

  • Hiển thị các database: SHOW DATABASES;
  • Tạo database mới: CREATE DATABASE ten_database;
  • Sử dụng database: USE ten_database;
  • Xóa database: DROP DATABASE ten_database; 

3. Thao tác với Bảng (Table)

  • Hiển thị các bảng: SHOW TABLES;
  • Tạo bảng mới:

CREATE TABLE users (

    id INT AUTO_INCREMENT PRIMARY KEY,

    name VARCHAR(100),

    email VARCHAR(100)

);

  • Xem cấu trúc bảng: DESCRIBE ten_bang;
  • Xóa bảng: DROP TABLE ten_bang;
  • Đổi tên bảng: RENAME TABLE ten_cu TO ten_moi; 

4. Thao tác Dữ liệu (CRUD)

  • Thêm dữ liệu (Insert):

INSERT INTO users (name, email) VALUES ('Nguyen Van A', 'a@example.com');

  • Truy vấn dữ liệu (Select):

SELECT * FROM users; -- Lấy tất cả

SELECT name FROM users WHERE id = 1; -- Lấy có điều kiện

  • Cập nhật dữ liệu (Update):

UPDATE users SET email = 'new@example.com' WHERE id = 1;

  • Xóa dữ liệu (Delete):

DELETE FROM users WHERE id = 1;

 

5. Sao lưu và Khôi phục

  • Sao lưu (Export) database:

mysqldump -u root -p ten_database > backup.sql

  • Khôi phục (Import) database:

mysql -u root -p ten_database < backup.sql

 

Một số lưu ý

  • Thoát khỏi MySQL: quit hoặc exit
  • Nếu quên dấu ; ở cuối dòng, hãy xuống dòng và gõ ; để thực thi câu lệnh trước đó.
  • Câu lệnh SQL không phân biệt hoa thường, nhưng tên bảng/cột thì có thể có (tùy cấu hình hệ điều hành).
